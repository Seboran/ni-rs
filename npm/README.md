# @seboran/ni

This project is like <https://github.com/antfu/ni> but in rust, because why not?

## Install

```sh
npm i -g @seboran/ni
```

## Features

### Install package

Examples

```sh
ni
# npm i
# pnpm i
# bun i
```

```sh
ni -D vite
# npm i -D vite
# pnpm add -D vite
# bun add -D vite
```

### Run script

```sh
nr my_script
# npm run my_script
# pnpm my_script
# bun my_script
```
